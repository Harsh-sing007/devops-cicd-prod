# ⚙️ DevOps CI/CD Pipeline with Docker

A CI/CD pipeline implementation that automates the build, security analysis, and containerization of an application using **GitHub Actions** and **Docker**.

The pipeline triggers automatically on commits and pull requests, ensuring continuous integration, automated validation, and secure build workflows.

---

## 🚀 Features

* Automated CI pipeline using **GitHub Actions**
* Docker-based container build process
* **CodeQL static analysis** for security scanning
* Automated workflows triggered on commits and pull requests
* Scheduled security scans for vulnerability detection
* Automatic version tagging for release management
